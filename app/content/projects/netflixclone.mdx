Dit is een van mijn eerste serieuze projecten. Ik wilde heel graag zo snel mogelijk coole dingen bouwen. 
Daarom begon ik gelijk met het leren van React. Ik ging toen een andere website na bouwen, 
zodat ik de focus kon leggen op de ins en outs van React in plaats van de aesthetics.

Next.js heeft automatische image optimization. Dit is waarom ik later het project om heb gezet 
naar een Next.js project. Dit was een verstandige keuze, aangezien er op de website veel 
afbeeldingen geladen moeten worden.
